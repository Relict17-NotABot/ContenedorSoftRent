Pagina Web dedicada al desarrollo y distribución de software que automatiza procesos para emprendedores y empresas. Con un sistema de subscripciones que ofrece servicioes de administración y desarrollo de sistemas automatizados. Idea del sistema Se ofreceran al usuario diferentes planes que contendrán servicios para administrar desde un emprendimiento hasta varios departamentos de una empresas con sistemas automatizados que se adapten a la necesidad del usuario. Por otro lado el usuario puede definir su plan personalizado, marcando los servicios y sistemas que necesite especificamente para su problema.
